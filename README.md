Как собрать проект:
1) Установи wsl
2) Установи кросс-компилятор в wsl используя скрипт
3) Установи mingw
4) Установи make
5) Установи nasm
6) Установи qemu
7) В папке проекта в консоли выполни команду "make"

How to build it:
1) Install wsl
2) Install cross compiler using script
3) Install mingw
4) Install make
5) Install nasm
6) Install qemu
7) In root directory run "make" in console
